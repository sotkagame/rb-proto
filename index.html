Перевес 7,5% на каждую карту (×1,85 вместо честных ×2). На коротких сериях факт скачет, на дистанции выходит ~7,5–7,9% (потолок ×20 чуть добавляет в пользу кассы — на последней карте выплата усечена). Доход берётся с оборота, а не с депозита: чем дольше крутят свои звёзды, тем больше касса.

Server seed фиксируется заранее (виден только его хэш). Цвет каждой карты = HMAC-SHA256(server seed, client seed:номер). Колода «перемешивается» после каждой карты — каждая независима, счёт карт невозможен. После раунда раскрываем server seed → любой может пересчитать и убедиться, что карты не подкручены.
